Клавишные комбинации

Клавишная комбинация

Действие

Заметки

F1

Открыть Справку


Enter / Return

Подтвердить ввод

Ничего не произойдет, если поле ввода пусто или вычисление завершено

Esc

Очистить Поле Вычислений

Если есть отмеченный текст -  выбор отменится

Вверх (стрелка)

Перейти в Историю в Свободном состоянии*;

Вверх по Истории в Свободном состоянии*


Вниз (стелка)

Вниз по Истории в Свободном состоянии*


Del / Delete

Удалить (обычным способом) текст справа или выделенный текст в состоянии* Редактирования;

Удалить текущую запись из списка в Свободном состоянии*;

Очистить Поле Вычислений после завершения вычисления


Ctrl + Shift + C

Изменить состояние опции «Автокопирование результата»


Ctrl + E

Выйти из программы

Кнопка Закрыть сворачивает программу в Трей

Ctrl + P

Открыть окно Настройки


Ctrl + H

Открыть и закрыть Историю (переключение между видами Простой и с Историей)


Ctrl + Up

Открыть Историю (переключается на вид с Историей), если закрыта.


Ctrl + Down

Закрыть Историю (переключается на вид Простой), если открыта.


Ctrl + X

Вырезать текст


Ctrl + C

Копировать (обычным способом) выделенный текст;

Копировать выражение текущей записи в Истории в Свободном состоянии*, если нет выделений текста


Ctrl + R

Копировать результат текущей записи в Истории в Свободном состоянии*


Ctrl + V

Вставить из буфера обмена


Ctrl + S

Сохранить Историю

Сохранение происходит автоматически при выходе из программы

Ctrl + Backspace

Удалить все слева от курсора в состоянии* Редактирования;

Очистить Поле Вычислений после завершения вычисления


Ctrl + Delete

Удалить все справа от курсора в состоянии* Редактирования



* Смотр. Интерфейс - «Поле вычислений и История», для справки о состояниях.
