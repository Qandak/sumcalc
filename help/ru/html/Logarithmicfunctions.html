Логарифмические функции

log10( x )

возвращает логарифм числа x по основанию 10.

log2( x )

возвращает логарифм числа x по основанию 2.

log( x , y )

возвращает логарифм числа x по основанию y.

lg( x )

возвращает логарифм числа x по основанию 10.

ln( x )

возвращает натуральный логарифм числа x (по основанию e).
