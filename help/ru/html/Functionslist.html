Список функций

Общие математические функции

abs( x )

возвращает абсолютную величину x.

amn( x1 , x2 , x3 , ...xN )

возвращает Арифметическое Среднее всех аргументов.

bin( x )

возвращает двоичное представление x.

cbrt( x )

возвращает кубический корень из x.

ceil( x )

возвращает ближайшее большее целое число от x.

dd( degrees )

переводит угол в десятичном представлении в Градусы(°), Минуты('), Секунды(") в виде функции dms(градусы, минуты, секунды).

deg( radians )

переводит Радианы в Градусы.

dms( degrees , minutes , seconds )

переводит Градусы(°), Минуты(') и Секунды(") в десятичное представление.

dperc( old , new )

возвращает разницу в процентном соотношении между числами [old] и [new].

exp( x )

возвращает e в степени x.

fact( n )

возвращает факториал числа n. Выдает ошибку если n не целое или отрицательное. n ограничен максимальным числом 64.

floor( x )

возвращает ближайшее меньшее целое число от x.

gcd( x , y )

возвращает Наибольший Общий Делитель (НОД) чисел x и y. Если x и y больше нуля, то значение gcd( x , y ) - наибольшее целое число, которое делит x и y.

gmn( x1 , x2 , x3 , ...xN )

возвращает Геометрическое Среднее всех аргументов.

hex( x )

возвращает шестнадцатеричное представление x.

hmn( x1 , x2 , x3 , ...xN )

возвращает Гармоническое Среднее всех аргументов.

hyp( x , y )

возвращает Евклидову норму, длину вектора от начала координат до точки (x, y).

max( x1 , x2 , x3 , ...xN )

возвращает наибольшее значение из всех аргументов.

min( x1 , x2 , x3 , ...xN )

возвращает наименьшее значение из всех аргументов.

mod( dividend , divisor )

возвращает остаток от деления ( dividend / divisor ).

oct( x )

возвращает восьмеричное представление x.

pc( x , percent )

возвращает [percent] процентов x (смотр. также функцию perc).

perc( x , percent )

возвращает x увеличенный или (-) уменьшенный на [percent] процентов (смотр. также функцию pc).

pow( x , y )

возвращает x в степени y.

rad( degrees )

переводит Градусы в Радианы.

round( x , dig )

возвращает ближайшее целое от x, если dig пропущен, иначе - округленное на [dig] символов после запятой.

rt( x , N )

возвращает корень из x с основанием N.

sqrt( x )

возвращает квадратный корень от x.

sum( x1 , x2 , x3 , ...xN )

возвращает сумму всех аргументов.

Тригонометрические фенкции

acosh( x )

возвращает гиперболический арккосинус числа x.

asinh( x )

возвращает гиперболический арксинус числа x.

atanh( x )

возвращает гиперболический арктангенс числа x.

cosh( x )

возвращает гиперболический косинус числа x.

sinh( x )

возвращает гиперболический синус числа x.

tanh( x )

возвращает гиперболический тангенс числа x.

acos( x )

возвращает арккосинус числа x в радианах или градусах.

asin( x )

возвращает арксинус числа x в радианах или градусах.

atan( x )

возвращает арктангенс числа x в радианах или градусах.

atan2( y , x )

возвращает atan(y / x) в радианах или градусах.

cos( x )

возвращает косинус угла x (радианы или градусы).

sin( x )

возвращает синус угла x (радианы или градусы).

tan( x )

возвращает тангенс угла x (радианы или градусы).

Логарифмические функции

log10( x )

возвращает логарифм числа x по основанию 10.

log2( x )

возвращает логарифм числа x по основанию 2.

log( x , y )

возвращает логарифм числа x по основанию y.

lg( x )

возвращает логарифм числа x по основанию 10.

ln( x )

возвращает натуральный логарифм числа x (по основанию e).

Специальные функции

cdf( x )

возвращает Функцию распределения (CDF) стандартного нормального распределения.

erf( x )

возвращает функцию ошибки от значения x.

erfc( x )

возвращает дополнительную функцию ошибки от значения x.

gamma( x )

возвращает гамма-функцию значения x.

lgamma( x )

возвращает натуральный логарифм абсолютного значения гамма-функции при x.
