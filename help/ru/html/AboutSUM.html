О программе

SUM - удобный «однострочный» кросс-платформенный калькулятор с Историей.

Программа предоставляет простой и минималистичный интерфейс для составления вычислений произвольной длины и сложности.

Средства вычисления программы основаны на интерпретаторе Python (для версии Python смотр. О программе в главном Меню).
