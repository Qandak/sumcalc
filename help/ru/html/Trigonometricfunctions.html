Тригонометрические функции

acosh( x )

возвращает гиперболический арккосинус числа x.

asinh( x )

возвращает гиперболический арксинус числа x.

atanh( x )

возвращает гиперболический арктангенс числа x.

cosh( x )

возвращает гиперболический косинус числа x.

sinh( x )

возвращает гиперболический синус числа x.

tanh( x )

возвращает гиперболический тангенс числа x.

acos( x )

возвращает арккосинус числа x в радианах или градусах.

asin( x )

возвращает арксинус числа x в радианах или градусах.

atan( x )

возвращает арктангенс числа x в радианах или градусах.

atan2( y , x )

возвращает atan(y / x) в радианах или градусах.

cos( x )

возвращает косинус угла x (радианы или градусы).

sin( x )

возвращает синус угла x (радианы или градусы).

tan( x )

возвращает тангенс угла x (радианы или градусы).
