Специальные функции

cdf( x )

возвращает Функцию распределения (CDF) стандартного нормального распределения.

erf( x )

возвращает функцию ошибки от значения x.

erfc( x )

возвращает дополнительную функцию ошибки от значения x.

gamma( x )

возвращает гамма-функцию значения x.

lgamma( x )

возвращает натуральный логарифм абсолютного значения гамма-функции при x.
