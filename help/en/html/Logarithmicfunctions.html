Logarithmic functions

log10( x )

returns the base-10 logarithm of x.

log2( x )

returns the base-2 logarithm of x.

log( x , y )

returns the logarithm of x to the y base.

lg( x )

returns the base-10 logarithm of x.

ln( x )

returns the natural logarithm of x (to base e).
