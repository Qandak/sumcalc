General math functions

abs( x )

returns the absolute value of x.

amn( x1 , x2 , x3 , ...xN )

returns the Arithmetic Mean (average) of all arguments.

bin( x )

returns the binary representation of x.

cbrt( x )

returns the cube root of x.

ceil( x )

returns the ceiling of x - the smallest integer greater than or equal to x.

dd( degrees )

converts degrees in decimal value to Degrees(°), Minutes(') and Seconds(") in the form of dms(degrees, minutes, seconds) function.

deg( radians )

converts Radians to Degrees.

dms( degrees , minutes , seconds )

converts Degrees(°), Minutes('), Seconds(") to decimal value.

dperc( old , new )

returns the percentage change from old to new numbers.

exp( x )

returns e raised to the power x.

fact( n )

returns n factorial. Raises an error if n is not integral or is negative. n is limited by 64.

floor( x )

returns the floor of x - the largest integer less than or equal to x.

gcd( x , y )

returns the Greatest Common Divisor (GCD) of integers x and y. If either x or y is nonzero, then the value of gcd( x , y ) is the largest positive integer that divides both x and y.

gmn( x1 , x2 , x3 , ...xN )

returns the Geometric Mean of all arguments.

hex( x )

returns the hexadecimal representation of x.

hmn( x1 , x2 , x3 , ...xN )

returns the Harmonic Mean of all arguments.

hyp( x , y )

returns the Euclidean norm, sqrt(x*x + y*y), the length of the vector from the origin to point (x, y).

max( x1 , x2 , x3 , ...xN )

returns the largest item in arguments list.

min( x1 , x2 , x3 , ...xN )

returns the smallest item in arguments list.

mod( dividend , divisor )

returns the remainder after division.

oct( x )

returns the octal representation of x.

pc( x , percent )

returns the percent of x (see also perc function).

perc( x , percent )

returns x increased or (-) decreased by percent (see also pc function).

pow( x , y )

returns x raised to the power y.

rad( degrees )

converts Degrees to Radians.

round( x , dig )

if dig is omitted, returns the nearest integer of x, otherwise - rounded to dig (digits) after point.

rt( x , N )

returns the Nth root of x.

sqrt( x )

returns the square root of x.

sum( x1 , x2 , x3 , ...xN )

returns the Sum of all arguments.
