Shortcut keys

Shortcut keys

Keyboard Shortcut

Action

Notes

F1

Open the program help


Enter / Return

Validate an input

Do nothing, if the input field is empty or a calculation is done

Esc

Clear Calculation Field

If there is a text selection - cancel selection

Up

Move to History in Free state*;

Up in history list in Free state*


Down

Down in history list in Free state*


Del / Delete

Delete (in usual way) text to the right or selected text in Edit state*;

Delete current item in list (i.e History) in Free state*;

Clear Calculation Field after calculation is done


Ctrl + Shift + C

Change "Autocopy result" mode


Ctrl + E

Exit

Close button minimize program to system Tray

Ctrl + P

Open Preferences window


Ctrl + H

Open and close History (switch between Simple and History modes)


Ctrl + Up

Open History (switch to History mode), if closed.


Ctrl + Down

Close History (switch to Simple mode), if open.


Ctrl + X

Cut selected text


Ctrl + C

Copy (in usual way) selected text;

Copy expression of current item in History in Free state*, if there is no text selection


Ctrl + R

Copy result of current item in History in Free state*, if the item is an expression or an assignment


Ctrl + V

Paste from clipboard


Ctrl + S

Save History

It is always automatically saving on Exit

Ctrl + Backspace

Delete all to the left of the cursor in Edit state*;

Clear Calculation Field after calculation is done


Ctrl + Delete

Delete all to the right of the cursor in Edit state*



* See User Interface - "Calculation Field and History", for states.
