Operators

SUM is using less operators, than other calculators. The reason is that the core principal of SUM is using keyboard (especially Numeric keypad, if available) instead of mouse or touch-pad. If an operator seems to be missing, it most probably can be found in Functions list.

So available operators are:

Addition, Subtraction  -  + (plus),  - (minus),

Multiplication, Division  -  * (asterisk),  / (slash),

Power  -  ** (double asterisk),

Division without reminder  -  // (double slash).

A keyboard operator like percent (%) is not available in SUM, as it can be confusing in relatively complex expressions. Using pc, perc and dperc functions is more flexible and explicit.

Delimiters

Thanks to parentheses "( )", expressions can be nested in arbitrary depth. Though parentheses are very helpful, missing or extra parentheses will raise an error.

Use of a comma "," delimiter is limited by function scope - only arguments of functions are comma separated. A missing or extra comma will raise an error as well.
